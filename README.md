# Package with Both C++ and Swift

This project is for testing C++ and Swift in the context of a library. 

- Apple Swift version 5.9 (swiftlang-5.9.0.128.2 clang-1500.0.40.1)
- XCode 15, beta 6

code can be run via either 

```zsh
swift test #or
swift run moarcxx
```
